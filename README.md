# Coffee Shop Sales Analysis Dashboard

This repository contains an Excel-based analysis and interactive dashboard for a coffee shop's sales operations. It provides clear insights into the transaction patterns, product performance, and store-level metrics.

## Project Overview
The data showcases various metrics indicating sales, transactions, and product details across store locations. The main focus is an interactive dashboard that visualizes store performance effectively.

### Files Included:
- **`coffee shop sales.xlsx`**: The main Excel workbook containing the raw data, processed pivot tables, and the interactive dashboard layout.
- **`Coffee Shop Sales Analysis.pdf`**: A PDF report/export of the final dashboard for easy viewing and sharing.
- **`Screenshot 2024-02-15 143928.png`**: A snapshot of the dashboard visualizations.

## Key Insights
Based on the extracted transaction and product data:
- Track sales performance by **Date/Time** dimensions (Month, Day, Hour).
- Analyze **Product Categories** and specific product types (e.g., beverages, food).
- Monitor sales across different **Store Locations**.
- Transaction-level granularity (Transaction ID, Quantity, Unit Price).

## Dashboard Features
The Excel dashboard makes use of:
*   **Pivot Tables**: Summarizing the large transactional dataset.
*   **Pivot Charts**: Visualizing trends over time (e.g., daily sales, hourly traffic).
*   **Slicers**: Allowing users to interactively filter data by month, store location, or product category.

## Usage
Simply download or clone this repository and open `coffee shop sales.xlsx` using Microsoft Excel. Use the slicers on the Dashboard sheet to filter the views and explore the insights!
